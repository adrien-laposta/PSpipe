# Rational numbers

In julia rational numbers can be constructed with the `//` operator.
Lets define two rational numbers, `x` and `y`:

```@example test
# Define variable x and y
x = 1//3
y = 2//5
```

When adding `x` and `y` together we obtain a new rational number:

```@example test
z = x + y
```
