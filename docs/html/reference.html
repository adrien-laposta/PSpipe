pspy.so_map.healpix2car(map, template, lmax=None)[source]¶

@brief Project a HEALPIX so map into a CAR so map the projection will be done in harmonic space, you can specify a lmax to choose a range of multipoles considered in the projection. If the coordinate of the map and the template differ, a rotation will be performed @param template: the car template you want to project into @param lmax: the maximum multipole in the HEALPIX map to project @return the projected map in the template pixellisation and coordinates
